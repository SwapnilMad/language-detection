## Getting Started
Clone or download the project to your system

### Prerequisites
Python, Ionic, Node.js, Angular

### Running
<p>Start Flask server</p>
<pre>python detect.py</pre>

<p>Install the app on ios device</p>
<pre>ionic cordova run ios --device</pre>
